Yönetici Girişi

Panel, değişiklikleri doğrudan GitHub deposuna işler ve site otomatik yayınlanır. Giriş için bir GitHub kişisel erişim anahtarı (fine-grained token) gerekir.

  1. GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token
  2. Repository access: yalnızca aligokten/DWG-3D-Viewer
  3. Permissions → Repository permissions → Contents: Read and write
  4. Oluşan github_pat_… anahtarını aşağıya yapıştırın.
Gelişmiş: depo ayarları
